
# Daily Planner Generator (Markdown)

Create a Markdown day planner with priorities, time blocks, notes, and reflection.

## Usage
```bash
python daily_planner.py --date 2026-02-18 --out ./plans
# If --date is omitted, uses today
```

## Notes
- File is named `planner_YYYY-MM-DD.md` and placed in the output folder.
